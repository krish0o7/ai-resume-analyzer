# AI Resume Analyzer

An AI-based Resume Analyzer that compares a resume with a job description and calculates match percentage.

## Features
- Skill-based matching
- Match score calculation
- Missing skill detection
- Clean UI using Flask

## Tech Stack
- Python
- Flask
- HTML
- CSS

## How to Run

1. Clone repository
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
